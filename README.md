# electron-vue-start

## Project setup
```
vue create electron-vue2-starter
vue add electron-builder
npm - electron:build
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
